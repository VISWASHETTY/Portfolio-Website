Why this Redesign?

A little personal project to improve the user experience and give the UI design of SPOTIFY a modern look with improved functionalities. The new interface retains the familiar elements of the old Spotify UI while introducing innovative ways for users to connect with friends, host music parties, view friends' profiles, and add friends' playlists to their own connection.
